LOOPS

A loop is a piece of code in a program that automatically repeats. One complete execution of all the statements within a loop is called an 'iteration' or 'pass'.The length of a loop is controlled by a conditional test made within the loop. While the tested conition is true, the loop will continue- until the test-expression is found to be false when the loop ends.

Problem

The function which we have implemented earlier, addVBlock() and addHBlock(), will be implemented iteratively here. Instead of writing addVBlock() or addHblock() several times, you can use syntax like

repeat for i times addVBlock/addHblock()

The mentioned command written shall be repeated i times.
